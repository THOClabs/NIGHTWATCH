# NIGHTWATCH
Autonomous Maksutov-Newtonian Observatory System

## What is this?
Voice-controlled telescope observatory designed for central Nevada
dark sky installation. Combines Russian optics excellence with
modern automation and local AI inference on DGX Spark.

## Key Components
- Intes-Micro MN78 (178mm f/6 Mak-Newt)
- DIY harmonic drive German Equatorial Mount
- OnStepX controller (Teensy 4.1)
- Ecowitt WS90 weather station + AAG CloudWatcher
- DGX Spark for voice pipeline and automation
