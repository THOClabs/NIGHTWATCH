# Assets

This directory is for project images and visual documentation.

## Suggested Content

- **Renders** — CAD renders of mount design, enclosure concepts
- **Photos** — Build progress photos, component images
- **Diagrams** — Wiring diagrams, system architecture visuals
- **Screenshots** — Software interface screenshots, voice pipeline demos

## Usage

Reference images in documentation using relative paths:

```markdown
![Mount Design](docs/assets/mount-render.png)
```

## Guidelines

- Use descriptive filenames (e.g., `harmonic-drive-assembly.jpg`, not `IMG_1234.jpg`)
- Optimize images for web (reasonable file sizes)
- Include alt text when embedding in markdown
